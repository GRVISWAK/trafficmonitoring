================================================================================
üìä SLIDING WINDOW DETECTION
================================================================================

**How it works:**
1. Collect 10 consecutive requests
2. Extract 9 behavioral features
3. Run hybrid ML detection (rules + 4 models)
4. Broadcast result via WebSocket
5. Slide window to next 10 requests

**9 Features:**
- request_rate ‚Üí DDoS detection
- unique_endpoint_count ‚Üí Reconnaissance
- method_ratio ‚Üí HTTP anomalies
- avg_payload_size ‚Üí Data exfiltration
- error_rate ‚Üí Scanning/injection
- repeated_parameter_ratio ‚Üí SQL injection
- user_agent_entropy ‚Üí Bot detection
- avg_response_time ‚Üí Performance
- max_response_time ‚Üí Latency spikes

**Hybrid Detection (4 models + 5 rules):**
```
risk = 30% rules + 25% isolation + 30% logistic + 15% failure
```

================================================================================
üé¨ SIMULATION MODES
================================================================================

1. **NORMAL** - Typical traffic (low risk)
2. **RATE_SPIKE** - DDoS attack (>15 req/s)
3. **ERROR_BURST** - Scanning (>50% errors)
4. **BOT_ATTACK** - Automated bot (low entropy)
5. **LARGE_PAYLOAD** - Exfiltration (>5000 bytes)
6. **ENDPOINT_SCAN** - Reconnaissance (>20 endpoints)
7. **MIXED** - Multiple attack types

### Test LIVE MODE (PowerShell)
```powershell
# Generate 10 requests to fill window
1..10 | ForEach-Object {
  Invoke-RestMethod -Method POST `
    -Uri "http://localhost:8000/login" `
    -Body (@{username="user$_"; password="pass"} | ConvertTo-Json) `
    -ContentType "application/json"
  Start-Sleep -Milliseconds 100
}
```

### Test SIMULATION MODE (PowerShell)
```powershell
# Start rate spike simulation
Invoke-RestMethod -Method POST `
  -Uri "http://localhost:8000/simulation/start" `
  -Body (@{mode="rate_spike"; duration_seconds=30} | ConvertTo-Json) `
  -ContentType "application/json"

# Check stats
Invoke-RestMethod -Uri "http://localhost:8000/simulation/stats"

# Stop simulation
Invoke-RestMethod -Method POST `
  -Uri "http://localhost:8000/simulation/stop"
```
