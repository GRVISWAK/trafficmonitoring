================================================================================
TROUBLESHOOTING:
================================================================================

Q: "No CSIC files found"
A: Make sure you copied the .txt files to backend/datasets/ folder
   (NOT backend/datasets/raw/ or backend/datasets/processed/)

Q: "Normal and anomalous traffic are combined"
A: No problem! The script detects attacks automatically using patterns like:
   - SQL injection keywords (SELECT, UNION, DROP, etc.)
   - XSS patterns (<script, javascript:, etc.)
   - Path traversal (../, etc/passwd, etc.)
   - Command injection (exec, eval, system, etc.)

Q: "How do I verify it worked?"
A: After running process_csic.bat, check:
   - backend/datasets/processed/csic_features.csv should exist
   - You'll see output like "Successfully parsed X requests"
   - You'll see "Normal requests: X, Attack requests: Y"
